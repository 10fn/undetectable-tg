## "Невидимый" telegram-клиент для прохождения собесов
- Не виден при демонстрации экрана
- Находится поверх всех окон
- Управляется горячими клавишами
- Настраиваемая прозрачность
- Невидимый курсор - всегда стандартный указатель
- Настраиваемый масштаб
- Отсутствие иконки в доке и трее



## Установка
#### Wndows
- `npm i`
- `npm run bild:win`
- В папке `dist` появится папка `win-unpacked`. В ней будет исполняемый файл `undetectable-tg.exe`

#### MacOS
- `npm i`
- `npm run build:mac`
- В папке `dist` появится папка `mac-arm64`. В ней будет исполняемый файл `undetectable-tg.dmg`
- Запускаем и даем необходимые разрешения




## Быстрый старт
1. Запустите приложение**
2. Используйте `⌘Cmd/Ctrl+⇧Shift+T`** для показа/скрытия
3. Перемещайте окно** с помощью `⌘Cmd/Ctrl+⇧Shift+M`
4. Настройте прозрачность** через `⌘Cmd/Ctrl+⇧Shift+↑/↓`
5. Настройте масштаб с помощью `⌘Cmd/Ctrl + / ⌘Cmd/Ctrl -`
